## üöÄ Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn package manager

### Installation
```bash
git clone https://github.com/yourusername/ats-resume.git
cd ats-resume
npm install
```

### Environment Setup
```bash
# Create .env file
VITE_GROQ_API_KEY=your_groq_api_key_here
VITE_MOCK_MODE=false
```

### Development
```bash
npm run dev
```

### Deployment
```bash
npm run build
# Deploy to Vercel
```
